# TwitterAPI CLI

TypeScript-first CLI for TwitterAPI.io.

## Principles

- JSON-first output
- strict TypeScript
- stable command contracts
- latest Node.js LTS baseline
- clear split between public read endpoints and authenticated write endpoints

## Scripts

- `pnpm dev -- --help`
- `pnpm build`
- `pnpm test`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm check`
